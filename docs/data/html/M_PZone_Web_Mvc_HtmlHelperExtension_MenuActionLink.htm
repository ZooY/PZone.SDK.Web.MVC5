Метод формирует элемент меню в виде ссылки. Для текущего пункта меню устанавливается CSS-стиль "current".

Примечание об использовании

В Visual Basic и C# этот метод можно вызывать как метод экземпляра для любого объекта типа HtmlHelper. При вызове метода для экземпляра следует опускать первый параметр.
