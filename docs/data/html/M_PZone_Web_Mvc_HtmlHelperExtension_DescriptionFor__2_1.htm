Возвращаемое значение

Тип: MvcHtmlString
Метод возвращает описание из атрибута [!:DisplayAttribute], определенного для свойства модели.

Примечание об использовании

В Visual Basic и C# этот метод можно вызывать как метод экземпляра для любого объекта типа HtmlHelperTModel. При вызове метода для экземпляра следует опускать первый параметр. Дополнительные сведения см. в разделе Методы расширения (Visual Basic) или Методы расширения (Руководство по программированию в C#).
